

## Directory Contents

- [`training_scripts_and_models/`](./training_scripts_and_models/)  
  Contains all necessary files for training the models used for **Research Question 1**. This includes training scripts, configuration files, and the resulting trained model checkpoints.

- [`RRQ1_ReqSeek_evaluation.ipynb`](./RQ1_ReqSeek_evaluation.ipynb)  
  Jupyter notebook for evaluating the final ReqSeek model. It reproduces the evaluation results reported in the paper.

- [`RQ1_10fold_evaluation.ipynb`](./RQ1_10fold_evaluation.ipynb)  
  Jupyter notebook for evaluating the trained candidate models using 10-fold cross-validation. It reproduces the evaluation results reported in the paper.

- [`human_annotated_data`](./human_annotated_data)  
  Contains human-annotated ARID requirments

- [`RQ1_sample_size_estimation.ipynb`](./RQ1_sample_size_estimation.ipynb)  
  Contains sample size estimation for human annotation 

- [`RQ1_inter_rater_and_berry_style_evaluation.ipynb`](./RQ1_inter_rater_and_berry_style_evaluation.ipynb)  
  Contains exploratory human-model evaluations on annotated data
