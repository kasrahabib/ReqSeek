

## Directory Contents

- [`training_scripts_and_models/`](./training_scripts_and_models/)  
  Contains all necessary files for training the models used for **Research Question 1**. This includes training scripts, configuration files, and the resulting trained model checkpoints.

- [`RRQ1_ReqSeek_evaluation.ipynb`](./RQ1_ReqSeek_evaluation.ipynb)  
  Jupyter notebook for evaluating the final ReqSeek model to address part of **Research Question 1**. It reproduces the evaluation results reported in the paper.

- [`RQ1_10fold_evaluation.ipynb`](./RQ1_10fold_evaluation.ipynb)  
  Jupyter notebook for evaluating the trained candidate models using 10-fold cross-validation to address **Research Question 1**. It reproduces the evaluation results reported in the paper.
